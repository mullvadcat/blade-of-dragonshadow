## 1. 游戏名称候选

| 候选名                   | 含义 / 卖点                                                                     |
| ------------------------ | ------------------------------------------------------------------------------- |
| **《龙刃归鞘》**（首选） | "归鞘"即收刀，直接点题"拔刀易，收刀难"。霸气但克制，最契合主题。demo 已用此名。 |
| **《龙影九斩》**         | 以核心绝学命名，强动作向、辨识度高，适合做成系列 IP。                           |
| **《止戈》**             | "止戈为武"——武的最高境界是停止干戈。文气重，适合独立游戏调性。                  |
| **《刀由心生》**         | 直指师父台词"刀由心控"，强调心法 > 刀法。                                       |
| **《赤怒青守》**         | 以双龙双色（赤=怒、青=守）命名，视觉与玩法系统（戾气/守心）双关。               |
| 《拔刀者》               | 偏冷硬、克制的命名，留白多。                                                    |
| 《雨夜断刃》             | 偏氛围向，适合做副标题或第一章名。                                              |

**推荐组合**：主标题《龙刃归鞘》 + 副标题 _Blade of the Returning Dragon_ / _To Sheathe the Dragon_。

## 2. 一句话卖点

> **一把为复仇出鞘的刀，最难的不是斩向仇人，而是在血海中学会何时收回。**

Steam 短描述：

> 暗黑东方武侠动作 RPG。继承父亲的龙刃刀与禁绝刀法"龙影九斩"，深入黑暗组织"黑鳞会"复仇。但你的每一次出刀——杀或不杀、护或不护——都会让刀身在赤红与青白之间偏移，最终决定你成为守护者，还是新的怪物。

## 3. 世界观设定

**时代**：架空东方武侠世界，王朝末年，秩序崩坏。表面上由名门正派维持江湖规矩，实则镖局、盐道、码头、赌坊、地下兵器市场早已被黑暗组织渗透。

**黑鳞会**：本作核心反派势力，兼具**江湖帮派**与**黑手党**双重属性。

- **明面**：经营赌坊、镖局、商路，控制盐铁等暴利行当。
- **暗面**：贩卖兵器、操纵官府、豢养杀手；更以**邪术**圈养、改造"妖物"作为私兵——这是它区别于普通帮派的恐怖之处。
- **组织结构**：会主沈无魇之下设"四鳞使"（分管杀手、商道、邪术、刑狱），底层为帮众、护卫、精英武士，邪术分支为术师与半妖化士兵。
- **信物**：黑鳞令牌——会徒身份与权限的象征，按鳞片数量分级。

**武侠为主，妖物为辅**：世界底色仍是人的江湖——门派、刀客、镖师、市井。妖物不是天生神魔，而是**黑鳞会用活人 + 邪术强行催化出的产物**，是"力量被滥用"的具象化恶果。因此妖物始终带着"人"的悲剧性，而非纯仙侠的法宝飞剑。

**龙影九斩的传说**：陆玄山所承的禁绝刀法。"一刀破甲、三刀断阵、九斩化龙"。它之所以被黑鳞会觊觎，因为**刀气能斩破邪术妖气**——是唯一能克制沈无魇妖物军团的力量。但越强的招式波及范围越大，这正是全作核心矛盾的源头。

## 4. 核心主题

**「少年因仇恨拔刀，最终为守护收刀。」**

三层递进：

1. **复仇（拔刀）**：少年陆云川因父亲被害而拔刀，被仇恨推动。
2. **成长 / 父子传承（识刀）**：在玄松老人引导与父亲死亡真相中，理解"刀"的两面。
3. **力量的克制（收刀）**：领悟父亲当年压制最强刀法以护村民的选择，从复仇者成长为守护者。

反派的镜像哲学：沈无魇代表"强者无需克制"，主角最终以"真正的强者，才有资格选择不伤害谁"回应。主题不是"杀死仇人就结束"，而是**力量必须被心驾驭**。

## 5. 主要角色设定

### 陆云川（主角）

- **身份**：陆玄山之子，少年刀客。
- **初始**：只会父亲教的基础刀法，没真正闯过江湖，被仇恨驱动。
- **弧光**：「我要杀光他们」→「我要阻止更多人被伤害」。
- **三阶段语气**：前期冲动愤怒、急于证明；中期在师父点拨下沉静；后期理解父亲，言语从"为什么"变为"我来守"。
- **玩法投影**：他的成长由**戾气值/守心值**系统外化——玩家的操作即角色的内心。

### 陆玄山（父亲，已故）

- **外号**：龙刃刀客。隐退的绝世高手，沉稳、克制、不炫武力。
- **死亡真相（核心反转）**：他**不是武功不敌**沈无魇。黑鳞会故意把战场设在村庄、以村民为人质——施展完整"龙影九斩"可全歼敌人但刀气会波及无辜，陆玄山遂**自我压制最强刀法，只以普通刀法护民**，最终被暗算身亡。
- **精神遗产**：真正留给儿子的不是秘籍，而是**"力量必须被心驾驭"**。贯穿全程以遗书、幻象、回忆碎片出现。

### 玄松老人（师父）

- **身份**：陆玄山的师父，隐居山林的老武者。
- **作用**：第二章末出现阻止失控的主角，第三章训练他，第五章并肩作战并重伤。
- **核心台词**：
  - "你现在不是在握刀，是被刀拖着走。"
  - "仇恨能让你拔刀，却不能教你何时收刀。"
  - "你父亲最强的一刀，不是斩出去的那一刀，而是忍住没有斩出去的那一刀。"

### 沈无魇（反派首领）

- **身份**：黑鳞会会主，地下江湖的统治者。冷酷、狡诈、野心极强。
- **力量哲学（不是单纯的坏）**：他出身底层，曾因"弱"而失去一切，由此笃信**"强者不必克制，克制只是弱者的借口"**。他夺取龙影九斩，是想同时掌控武林高手与妖物之力，建立"强者支配一切"的新秩序。在他眼里，陆玄山的"克制"是懦弱，是他必须证伪的命题。
- **悲剧性**：他和主角是同一道选择题的两个答案——这让最终对决成为价值观的决战，而非善恶脸谱。

### 赤鳞鬼（妖物 Boss）

- **身份**：黑鳞会以邪术 + 活人催化出的妖物，赤红鳞片、极强再生。
- **剧情作用**：象征"滥用力量的恶果"；是主角**第一次真正理解父亲"斩妖不伤人"刀法**的关键战。它体内残留着被试验者的人性碎片，让"杀它"本身也带着悲悯。

### 支线角色（建议补充）

- **阿绫**：黑石城被囚百姓中的少女，串联"保护 NPC"系统的情感锚点，三结局对她命运处理不同。
- **断鳞**：黑鳞会底层叛逃的帮众，灰色线人，可在守护线中被感化、复仇线中被抛弃。

## 6. 六章主线剧情

> 每章含：剧情目标 · 主要场景 · 核心玩法 · 敌人 · Boss · 章节结尾。

### 第一章：雨夜疑案 _（即现有 demo 的垂直切片）_

- **目标**：调查父亲死亡真相。
- **场景**：村庄、父亲旧宅、祠堂、雨夜山路、父亲书房、地下密室。
- **玩法**：调查取证、对话、基础移动、简单战斗、机关解谜。集齐四条线索（伤痕 / 黑鳞令牌 / 烧灼刀鞘 / 被威胁村民）方可开启书房暗门。
- **敌人**：黑鳞会探子、伪装山匪的杀手。
- **Boss**：黑鳞会刺客 **乌针**（针刺/烟遁两阶段，对应 demo 已实现）。
- **结尾**：密室获《龙影刀经》、龙刃刀、父亲遗书。少年被仇恨吞没，带刀离家复仇。

### 第二章：龙刃初鸣

- **目标**：离村追踪黑鳞会线索。
- **场景**：荒废驿站、竹林小道、山神庙、断桥。
- **玩法**：系统化刀法教学（轻/重/闪避/格挡），首次精英战；引入**戾气值**的第一次显现。
- **敌人**：黑鳞帮众、弩手、双刀杀手。
- **Boss**：小头目 **铁臂罗**（重甲+抓投，教学"破防"）。
- **结尾**：主角险胜，但因愤怒失控**差点误伤无辜路人**——龙刃首次泛红。玄松老人现身阻止，带他离开。**（"力量克制"主题的第一次玩法化警示）**

### 第三章：七日练刀

- **目标**：玄松老人训练主角，让他真正理解龙影刀法。
- **场景**：山谷、瀑布、竹林、石阵、悬崖、梦境幻境。
- **玩法**：训练关 + 能力解锁 + 技能试炼 + 心魔挑战。**七日 = 七个主题小关**：

| 日        | 主题             | 解锁                       |
| --------- | ---------------- | -------------------------- |
| 一 · 练气 | 提升体力上限     | 体力/耐力基础              |
| 二 · 练步 | 闪避与突进       | 闪避、潜龙出渊（第一斩）   |
| 三 · 练势 | 连击与收招       | 连段、游龙回身（第二斩）   |
| 四 · 练守 | 格挡与完美格挡   | 格挡、听风断影（第四斩）   |
| 五 · 练破 | 破甲与打断       | 破防、裂鳞破甲（第三斩）   |
| 六 · 练意 | 刀气与蓄力斩     | 刀气、苍龙吐息（第五斩）   |
| 七 · 练心 | 面对父亲死亡幻象 | 理解"刀由心控"，解锁守心值 |

- **Boss**：**主角的心魔幻影**（手持泛红龙刃的"另一个自己"，越用戾气招式它越强）。
- **结尾**：初步领悟龙影九斩前几式，但仍未真正放下仇恨——心魔被击退而非消灭。

### 第四章：黑石城

- **目标**：与玄松老人潜入黑鳞会大本营黑石城。
- **场景**：外城、赌坊、兵器库、杀手训练营、地下牢房。
- **玩法**：潜入 × 战斗混合；**救出被囚村民**（守心值核心来源）；摧毁据点。
- **敌人**：帮众、重甲护卫、毒药刺客、弩箭队、黑鳞精英武士。
- **Boss**：黑鳞最强士兵 **屠山**（纯粹暴力的化身，重兵器，象征"力量不加节制"的极端）。
- **结尾**：击败屠山，却发现牢中关押大量百姓与**被试验的妖物**——黑鳞会的真正恶行浮出水面。

### 第五章：赤鳞鬼牢

- **目标**：黑鳞会释放妖物赤鳞鬼，主角与玄松老人联手阻止灾难。
- **场景**：地下妖牢、血池、锁妖台、崩塌地宫。
- **玩法**：双人协作剧情战、Boss 多阶段、**保护无辜 NPC + 限制场地破坏**（首次把"克制"做成硬性战斗机制）。
- **敌人**：半妖化士兵、邪术师、妖化黑鳞护卫。
- **Boss**：**赤鳞鬼**（四阶段，详见第 10 节）。
- **结尾**：玄松老人为护人重伤。主角顿悟：**最难的不是杀死敌人，而是在混乱中保护弱者**。

### 第六章：龙影终章

- **目标**：主角独闯黑鳞总殿，对决沈无魇。
- **场景**：黑鳞总殿、燃烧城楼、龙纹祭坛、最终大殿。
- **玩法**：最终多阶段 Boss + 剧情抉择 + 终极奥义解锁。
- **Boss**：**沈无魇**（四阶段，详见第 10 节）。
- **关键反转**：主角得知**父亲当年面对过完全相同的选择**。无视百姓强行释放完整九斩→速胜但伤亡惨重（导向复仇/黑化）；克制刀法保护百姓→进入更难的真正终局（导向守护结局）。
- **结尾**：领悟真正的"龙影九斩"非毁灭，而是**收放自如、斩恶不伤人**。报父仇，并继承父亲守护江湖的意志（视分支而定）。

## 7. 战斗系统设计

**类型定位**：**类魂轻量化 2.5D 横版动作 RPG**——体力管理 + 完美格挡 + 破防的硬核手感，但难度曲线友好、关卡线性精炼，适合独立团队与 Demo 落地（现有 demo 已是 2D 横版 Phaser 框架，天然契合）。

### 基础机制

1. **普通攻击（轻斩）**：快速连段，收招快，破防低。
2. **重攻击（重斩）**：高伤、高破防，前摇大、可被打断。
3. **闪避**：消耗体力，附短暂无敌帧。
4. **格挡**：抵挡正面攻击，按住消耗体力，背后/破防招无法格挡。
5. **完美格挡**：命中瞬间（窗口 ~140ms）格挡，免伤并开启**反击窗口**（接"听风断影"打断敌技）。_（demo 已实现该机制与反击窗口）_
6. **破防值（架势/护甲）**：连击、重击、特定刀法可破；破防后敌人硬直，可接处决/重斩。
7. **刀气**（中期解锁）：远距离斩击，消耗龙魂，破邪术/远程压制。
8. **龙魂值**：通过战斗、完美格挡、保护 NPC、击败精英积累，是九斩与刀气的能量池。
9. **龙影九斩**：核心绝学，分阶段解锁；技能形态随戾气/守心偏移（见下）。
10. **克制系统（本作灵魂）**：
    - **戾气值 ↑**：连续杀戮、忽视/误伤 NPC、滥用大范围 & 处决招、残酷对话。
    - **守心值 ↑**：保护百姓、非致命解决、克制使用范围杀招、放过可饶恕之敌。
    - **双值互相拉扯**（不是单一滑条），影响：**剧情分支、技能形态、刀身颜色、结局**。

### 双值如何"可玩"

- **技能形态切换**：同一招九斩，高戾气=红色狂暴形态（更高伤害、更大波及、附带反噬/伤己或误伤判定）；高守心=青白守护形态（范围可控、附护盾/控制/救援判定）。
- **环境反馈**：高戾气时刀身泛红、敌人更具攻击性、NPC 畏惧；高守心时刀身青白、可触发 NPC 协助、隐藏对话。
- **硬机制关卡**（第五、六章）：在"输出 vs 保护"间二选一的实时抉择，直接改写战斗目标与 Boss 阶段。

## 9. 龙刃刀设定

**外观**

- 刀身：龙鳞纹路，刀背暗沉厚重，刀锋青白寒光。
- 刀柄：缠黑色皮革，护手形如龙爪。
- 刀鞘：黑金色，刻双龙——**怒龙**（象征复仇）与**守龙**（象征克制）盘绕缠斗。

**特殊能力**

1. **感应妖气**：靠近邪术/妖物时刀身微鸣、纹路发亮（兼作探索引导）。
2. **吸收龙魂值**：击杀/完美格挡/护民时积蓄能量。
3. **与九斩共鸣**：解锁招式时刀纹逐式点亮。
4. **怒态**：主角被仇恨支配（高戾气）时刀身泛**红**，伤害↑但易失控、误伤判定开启。
5. **守态**：主角守住本心（高守心）时刀身泛**青白**，招式可控、附守护增益。
6. **终末觉醒**：真结局中龙刃不再是复仇之刀，而成**守护之刀**——双龙由"绞斗"化为"共舞"。

> 龙刃刀本身就是一根**可视化的道德进度条**：玩家不看 UI，只看刀的颜色，就知道自己正在变成谁。

## 10. Boss 设计

| Boss         | 章  | 定位 / 主题                        | 招式 & 阶段                                                                                                                                             |
| ------------ | --- | ---------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **乌针**     | 一  | 教学型刺客，"看不见的刀"           | ①针刺突进（练闪避） ②烟遁狂暴+追击（练读招）。_demo 已实现两阶段_                                                                                       |
| **铁臂罗**   | 二  | 重甲抓投，教"破防"                 | 重拳/抓投/铁臂格挡；须用裂鳞破甲破其架势                                                                                                                |
| **心魔幻影** | 三  | 价值观试炼，"另一个你"             | 复制玩家招式；**玩家戾气越高它越强**，守心越高它招式越钝——以玩法叙事                                                                                    |
| **屠山**     | 四  | 纯粹暴力的极端                     | 巨刃横扫/震地/不可格挡的蓄力；强调闪避与处决窗口                                                                                                        |
| **赤鳞鬼**   | 五  | 滥用力量的恶果，"斩妖不伤人"的考验 | ①利爪冲撞撕裂 ②吞妖气狂暴、范围扩大 ③**冲向被囚百姓——玩家须在输出与救援间抉择** ④与玄松合击斩杀                                                         |
| **沈无魇**   | 六  | 终局，力量哲学决战                 | ①人形武者：掌法/暗器/刀法 ②邪术：黑鳞幻影+妖气锁链 ③半妖：融妖力、范围剧增 ④**终局：挟持百姓逼你使出完整九斩**——克制则进真正终局，无视则速胜入复仇/黑化 |

**Boss 设计原则**：每个 Boss 都对应一项战斗教学 + 一层主题。最终 Boss 不是"血更厚的怪"，而是**让玩家亲手做出父亲当年那道选择题**。

## 11. 结局设计

| 结局                   | 触发条件                                                     | 结果                                                                                   | 主题                                           |
| ---------------------- | ------------------------------------------------------------ | -------------------------------------------------------------------------------------- | ---------------------------------------------- |
| **复仇结局**           | 戾气偏高、救人较少，但未完全黑化；终局选择强行释放刀法       | 杀死沈无魇，黑石城伤亡惨重。心未得安宁，带刀独行，成为孤独刀客                         | **仇报了，心却没被救回来**                     |
| **守护结局**（真结局） | 守心值高、多次护民、克制范围杀招；终局选择保护百姓入更难终局 | 击败沈无魇并救下百姓，瓦解黑鳞会。玄松认可他真正继承父亲的刀，主角继续行走江湖守护弱者 | **真正继承父亲的，不是刀法，而是心法**         |
| **黑化结局**           | 戾气极高、频繁逆鳞断魂/怒形态、忽视 NPC、残酷路线            | 杀死沈无魇却被仇恨吞噬，龙刃彻底变红，主角成为新的恐惧——"血龙刀客"                     | **当复仇成为唯一目的，人会变成自己最恨的怪物** |

> 三结局共用同一场最终战，但**因玩家全程行为（戾气/守心累积）+ 终局一次抉择**自然分流，避免"结局选项菜单"的廉价感。

## 12. 美术风格

**总基调**：东方暗黑武侠。写实剪影 + 戏剧化光影，水墨笔触与颗粒质感融合；妖物为辅、人间为主，避免仙侠飞剑霓虹感。

**色彩语言（贯穿全作的视觉语法）**

- 🔴 **红**：仇恨、妖气、失控（赤鳞、妖牢血池、怒态刀身）。
- 🔵⚪ **青白**：克制、守护、父亲传承（龙刃寒光、守态、回忆与遗物）。
- ⚫🟡 **黑金**：龙刃刀与江湖旧时代的厚重（刀鞘、密室机关、总殿）。

**分项**

- **场景**：雨夜村庄、旧宅密室烛火机关、竹林山谷瀑布修炼、黑石城赌坊与兵器库、地下妖牢锁妖台、破败燃烧的龙纹大殿。
- **角色**：主角青白冷色基调，戾气上升时逐渐渗入红；父亲幻象半透明青白；沈无魇黑金华服 + 半妖阶段渗赤鳞。
- **武器**：龙刃刀为美术核心资产，双龙刀鞘 + 随状态变色的刀身。
- **敌人**：人类敌人写实武侠装束；妖物保留"人形残影"以强化悲剧感（赤鳞下隐约的人脸/手）。
- **UI**：水墨卷轴 + 黑金边框；血/体力/龙魂三条；**戾气-守心不用数字条，而用刀身颜色 + 画面边缘红/青晕染**呈现。
- **技能特效**：刀气如龙形青白光痕；怒形态拖拽赤红碎鳞与血雾；守形态化为环绕式青白光罩。核心视觉冲突=**赤怒 vs 青守**。

## 13. 开场 CG 旁白

> _（画面：雨夜，村口新坟。镜头推近泥水中半埋的黑鳞令牌。）_

雨，下了一整夜。
没有人记得，刀客也是会死的。

> _（旧宅，烛火。少年的手抚过父亲冰冷的刀鞘，指尖触到鞘上烧灼的焦痕。）_

他们说，他是失足坠崖。
可父亲的伤口太直、太静——
那不是山石的手笔，是黑夜里一柄短刃的耐心。

> _（书房深处，机括低响，暗门裂开一线，潮湿的铁锈味涌出。）_

墙后，藏着他从不肯说的另一半人生：
一本刀经，一把刀，一封信。

> _（龙刃出鞘，青白寒光劈开烛影，刀身龙鳞流转。）_

"刀，可以杀人，也可以救人。
若有一日你拔出龙刃——
记住，不要让仇恨，替你握刀。"

> _（少年合上遗书，眼底却只剩火。他背刀，推门，走进雨里。）_

他还不懂这句话。
他只知道，有些人，该用这把刀去找。

> _（雨幕吞没少年背影，刀鞘上的双龙——一条怒，一条守——在闪电里一瞬交错。）_

少年因仇恨拔刀。
至于何时收刀……那是后来，很久以后，他才学会的事。

_（标题浮现：**龙刃归鞘**。）_

## 14. 可扩展 DLC / 续作方向

1. **前传 DLC《龙刃刀客》**：操作年轻时的**陆玄山**，讲述他从顶级杀手到"见过太多杀戮而归隐"的转变，以及村庄那一夜他亲历的同一道选择题——与正作终章形成跨代回响。
2. **支线 DLC《七日之外》**：扩展玄松老人的过往与"龙影九斩"的真正来历（是否还有被刻意封印的"第十斩"）。
3. **续作《龙刃·江湖卷》**：守护结局后的开放江湖。主角作为新一代"龙刃刀客"行走各地，铲除黑鳞会残党与新生妖患，引入门派系统、声望与"传承弟子"机制——把"收刀"主题延伸为"教人收刀"。
4. **挑战模式 / Roguelite 副本《心魔回廊》**：以第三章心魔幻影为核心的轮抽式连战，随戾气/守心解锁不同九斩形态组合，提供长线 build 深度。
5. **黑化线续作钩子《血龙》**：若黑化结局达成度高，开放一条"以恶制恶"的反英雄战役，作为道德实验性内容。
