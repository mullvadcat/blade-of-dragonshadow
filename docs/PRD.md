## 1. 产品定位

### 1.1 一句话定位

一款**暗黑东方武侠动作 RPG**，围绕"少年因仇恨拔刀，最终为守护收刀"展开，把"力量的克制"做成可玩的核心系统。

### 1.2 题材边界（硬约束）

- ✅ **武侠为主，妖物为辅**：世界底色是人的江湖（门派、刀客、镖局、市井黑帮）。
- ✅ **妖物是恶果而非神魔**：妖物由"活人 + 邪术"催化而成，是"力量被滥用"的产物，带悲剧性。
- ❌ **不做纯仙侠 / 修真**：无飞剑、御空、渡劫、法宝体系。
- 🎨 **整体风格**：暗黑东方武侠，氛围沉郁，写实剪影 + 水墨质感。

### 1.3 候选游戏类型（任选其一落地）

- 2D 横版动作武侠（**现 demo 采用，推荐**）
- 俯视角动作 RPG
- 类魂轻量化动作 RPG

## 2. 核心主题与价值

**主题句**：「少年因仇恨拔刀，最终为守护收刀。」

三层叙事递进，必须在玩法与剧情中均有体现：

1. **复仇（拔刀）** — 动机起点。
2. **成长 / 父子传承（识刀）** — 中段转折。
3. **力量的克制（收刀）** — 终极立意。

**设计红线**：

- 主题落点**不是**"杀死仇人即结束"，而是"力量必须被心驾驭"。
- 主角成长弧光必须清晰：「我要杀光他们」→「我要阻止更多人被伤害」。
- 反派**不能脸谱化**，必须有自洽的力量哲学（"强者不必克制"）。
- 父亲死亡真相**必须有反转**（见 §4）。

---

## 3. 世界观需求

- **黑鳞会**：核心反派组织，兼具江湖帮派与黑手党属性。控制赌坊、盐道、码头、地下兵器市场；操纵官府、豢养杀手；以邪术豢养妖物作为私兵。信物为"黑鳞令牌"。
- **龙影九斩**：禁绝刀法，"一刀破甲、三刀断阵、九斩化龙"。关键设定——**刀气能斩破邪术妖气**，是唯一克制黑鳞会妖物军团的力量；但招式越强，波及范围越大（核心矛盾源）。
- **龙刃刀**：传世宝刀，双龙刀鞘（怒龙=复仇 / 守龙=克制），随主角心境变色（详见 GDD §9）。

## 7. 战斗系统需求

### 7.1 基础机制（P0）

轻攻击 / 重攻击 / 闪避（含无敌帧）/ 格挡 / **完美格挡（触发反击）** / 破防值（架势·护甲）。

### 7.2 进阶机制（P1）

- **刀气**：中期解锁，远程斩击。
- **龙魂值**：能量池，由战斗、完美格挡、保护 NPC、击败精英积累。
- **龙影九斩**：分阶段解锁的核心绝学，技能形态随道德值偏移。

### 7.3 克制系统（P0，本作差异化核心）

- **戾气值**：连续杀戮、忽视/误伤 NPC、滥用大范围与处决招、残酷对话 → ↑
- **守心值**：保护百姓、非致命解决、克制范围杀招、放过可饶者 → ↑
- **影响维度**：剧情分支、技能形态、刀身颜色、最终结局。
- **可视化要求**：道德倾向优先通过**刀身颜色 + 画面边缘红/青晕染**呈现，弱化裸数字条。

详见 GDD §7。

## 8. 龙影九斩需求

九式技能，每式须定义：名称 · 效果 · 解锁章节 · 进攻用途 · 防守/守护用途。
方向锚点：潜龙出渊 / 游龙回身 / 裂鳞破甲 / 听风断影 / 苍龙吐息 / 盘龙守心 / 双龙绞阵 / 逆鳞断魂 / 天龙归刃。
第九式"天龙归刃"须有**怒（毁灭）/ 守（守护）双形态**。完整表见 GDD §8。

## 10. 结局需求

三结局，由全程戾气/守心累积 + 终局抉择自然分流（**禁止结局菜单式选项**）：

1. **复仇结局** — 仇报了，心没被救回来。
2. **守护结局**（真结局） — 真正继承的是心法而非刀法。
3. **黑化结局** — 复仇成唯一目的，人变成自己最恨的怪物。

条件与文案见 GDD §11。

## 13. 验收基线（质量红线）

- [ ] 剧情紧凑、画面感强。
- [ ] 主角成长弧光清晰可感。
- [ ] 父亲死亡真相有反转。
- [ ] 反派有自洽力量哲学，非纯粹的恶。
- [ ] 武侠为主、妖物为辅，无纯仙侠修真元素。
- [ ] "力量的克制"主题贯穿剧情与玩法（戾气/守心系统）。
- [ ] 三结局由玩家行为自然分流，而非选项菜单。
- [ ] 可落地为动作 RPG / 2D 横版武侠 / 独立游戏 Demo。
